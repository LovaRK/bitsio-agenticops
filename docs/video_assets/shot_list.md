# Ready-To-Record Shot List (90s, exact app flow)
Use app at `http://127.0.0.1:3000`.

1. **00:00-00:08**
   - Start on **Dashboard**. Slow zoom in KPI cards.
   - Cursor hover KPI + runtime badge.
   - Purpose text overlay: “From noisy telemetry to explainable decisions.”

2. **00:08-00:18**
   - Click **Incidents** (left nav).
   - Open one incident detail.
   - Scroll timeline nodes: evidence, correlation, reasoning, confidence, approval.
   - Hover a tool chip to show inline details.

3. **00:18-00:28**
   - Show **Decision Gate** area.
   - Type short approval note.
   - Hover **Approve** / **Reject** buttons (don’t actually submit if demo data must stay unchanged).
   - Overlay: “Human-in-the-loop guardrail.”

4. **00:28-00:38**
   - Click **Approvals** tab.
   - Show pending queue + incident links.
   - Brief highlight on auditability/traceability text.

5. **00:38-00:50**
   - Click **Monitoring**.
   - Show uptime, latency, active nodes, runtime health.
   - Hover “How calculated” blocks.

6. **00:50-01:04**
   - Click **Telemetry Value Impact**.
   - Focus on Annual Spend, Potential Savings, Utilization, Security gaps.
   - Scroll to recommendations + value matrix.
   - Overlay: “Business impact from real telemetry.”

7. **01:04-01:16**
   - Click **Fraud Risk Agent**.
   - Show risk score, active cases, policy evaluation, agent telemetry.
   - Highlight explainability + compliance section.

8. **01:16-01:26**
   - Click **Settings**.
   - Show runtime mode switcher (Local Dev / Local Integration / Cloud Model Test / Cloud Live).
   - Click **Test Connections** and show status alert.

9. **01:26-01:30**
   - Return to Dashboard hero view.
   - Final overlay: “BitsIO AgenticOps: Faster response. Lower waste. Safer decisions.”

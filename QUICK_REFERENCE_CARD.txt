❓ QUICK TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  White screen?
  └─ Cmd+Shift+Delete (clear cache), reload

  Port 3000 in use?
  └─ lsof -i :3000 | grep LISTEN | awk '{print $2}' | xargs kill -9

  Docker not starting?
  └─ colima start (or open Docker Desktop app)

  Need API working?
  └─ API fallback mock data is enabled; you're seeing realistic demo data

  Modified mock data and need restart?
  └─ docker-compose restart api
